# Aditya Portfolio Pro+
React + Vite + Tailwind + Framer Motion + SEO + Blog + Admin (client-side JSON editor).

## Run
npm install
npm run dev

## Edit content
Use `/admin` to import/export JSON. Replace `src/data.js` with exported JSON or maintain outside.

## Contact form
Replace the Formspree endpoint in Home.jsx form action: `https://formspree.io/f/your-form-id`.
